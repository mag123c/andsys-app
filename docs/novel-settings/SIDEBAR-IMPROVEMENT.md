# 사이드바 개선

## 개요

소설 상세 페이지(`NovelSidebar`)의 UX 개선.

## 현재 구조

```
┌─────────────────────┐
│ ← 소설 목록          │
│ 소설 제목            │
├─────────────────────┤
│ 📖 회차 목록         │  ← 모두 동일 색상
│ 📄 시놉시스          │
│ 👥 등장인물          │
│ 🔗 관계도            │
├─────────────────────┤
│ 회차 (N)            │  ← 회차 페이지에서만 표시
│   EP.01 제목 1,234자 │
│   EP.02 제목 2,345자 │
│   ...               │
├─────────────────────┤
│ N개 회차 · 총 글자수  │
└─────────────────────┘
```

## 변경 사항

### 1. 중간 "회차" 섹션 제거

- 현재: 회차 페이지/편집 중일 때 중간에 회차 목록 표시
- 변경: 제거 → 꺾쇠로 통합

### 2. 꺾쇠(Chevron) 펼침/접힘

**대상 메뉴:**
- 회차 목록 (리스트: 회차들)
- 등장인물 (리스트: 캐릭터들)

**동작:**
| 클릭 영역 | 동작 |
|----------|------|
| 메뉴 텍스트 | 해당 페이지로 이동 |
| 꺾쇠 아이콘 | 리스트 펼침/접힘 |
| 펼친 리스트 항목 | 해당 상세로 이동 |

**자동 펼침:**
- 해당 페이지 진입 시 자동 펼침
- 예: `/novels/[id]` → 회차 목록 자동 펼침
- 예: `/novels/[id]/characters` → 등장인물 자동 펼침

**빈 상태:**
- 회차 0개 → 꺾쇠 숨김
- 등장인물 0명 → 꺾쇠 숨김

**상태 저장:**
- 새로고침 시 초기화 (저장 안 함)

**최대 표시:**
- 20개까지 표시
- 초과 시 "더보기" 버튼

### 3. 리스트 표시 내용

**회차 목록:**
```
EP.01 회차 제목     1,234자
EP.02 회차 제목     2,345자
```

**등장인물:**
```
홍길동
김철수
이영희
```
- 이름만 표시 (이미지, 역할 없음)

### 4. 메뉴별 색상

각 메뉴에 고유 색상 적용 (눈에 편한 톤):

| 메뉴 | 색상 제안 |
|------|----------|
| 회차 목록 | Blue (slate-500) |
| 시놉시스 | Green (emerald-500) |
| 등장인물 | Purple (violet-500) |
| 관계도 | Rose (rose-500) |

**적용 위치:**
- 아이콘 색상
- 활성 상태 배경색 (연한 톤)

## 변경 후 구조

```
┌─────────────────────┐
│ ← 소설 목록          │
│ 소설 제목            │
├─────────────────────┤
│ 📖 회차 목록      ▼  │  ← 파랑, 꺾쇠
│   EP.01 제목  1,234자│
│   EP.02 제목  2,345자│
│   더보기...          │
│ 📄 시놉시스          │  ← 초록
│ 👥 등장인물       ▶  │  ← 보라, 꺾쇠
│ 🔗 관계도            │  ← 로즈
├─────────────────────┤
│ N개 회차 · 총 글자수  │
└─────────────────────┘
```
